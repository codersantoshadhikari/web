    Dart Conditions and Loops

    Dart Programming: Control Structures and Exception Handling#

    Dart is a popular language used for developing mobile, desktop, and web applications. Understanding its control structures and exception handling mechanisms is crucial for effective coding. Here’s an overview of the key elements:

    1. Conditions in Dart#

    Dart uses the if and else statements for conditional execution:

    int number = 10;
if (number % 2 == 0) {
  print('Even');
} else {
  print('Odd');
}
    1. Loops in Dart For Loop The for loop in Dart is used to repeat a block of code a specific number of times:
    for (int i = 0; i < 5; i++) {
  print('Iteration $i');
}

    For Each Loop Dart supports iterating over collections using the for-in loop:

    List<String> names = ['Alice', 'Bob', 'Charlie'];
for (String name in names) {
  print(name);
}

    While Loop The while loop repeats a block of code as long as the condition is true:

    int num = 5;
while (num > 0) {
  print(num);
  num--;
}

    Do While Loop The do-while loop is similar to the while loop but ensures the code block is executed at least once:

    int num = 5;
do {
  print(num);
  num--;
} while (num > 0);
    1. Switch Case in Dart Dart uses the switch statement for handling multiple conditional branches:
    String grade = 'B';
switch (grade) {
  case 'A':
    print('Excellent');
    break;
  case 'B':
    print('Good');
    break;
  default:
    print('Ungraded');
}
    1. Break and Continue in Dart break is used to exit the loop immediately. continue skips the rest of the loop iteration:
    for (int i = 0; i < 10; i++) {
  if (i == 5) {
    continue;  // Skip iteration 5
  }
  print(i);
}
    1. Ternary Operator in Dart The ternary operator provides a shorthand for if-else:
    int x = 100;
String result = x > 50 ? 'High' : 'Low';
print(result);
6. Exception Handling in Dart
Dart handles exceptions with try, catch, and finally blocks:



```dart

try {
  int result = 100 ~/ 0;
} on IntegerDivisionByZeroException {
  print('Cannot divide by zero');
} catch (e) {
  print('Exception occurred: $e');
} finally {
  print('Process complete');
}
