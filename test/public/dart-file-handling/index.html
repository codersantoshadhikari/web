    Dart File Handling

    Dart File Handling#

    Dart provides robust support for file handling, enabling operations such as reading, writing, and manipulating files and directories. This section covers various aspects of file handling in Dart.

    1. Introduction to File Handling in Dart#

    • Overview: Dart utilizes the dart:io library to perform file operations. This library includes functionalities for reading and writing files, managing directories, and handling file paths.

    2. Reading Files#

    • Synchronous File Reading: Reads the entire file content in a blocking manner, halting execution until the read operation completes.

      Example:

      import 'dart:io';

void main() {
  String content = File('example.txt').readAsStringSync();
  print(content);
}

    • Asynchronous File Reading: Reads the file content asynchronously, allowing other operations to proceed without waiting for the read operation to finish.

    Example# 

    import 'dart:io';

void main() async {
String content = await File('example.txt').readAsString();
print(content);
}
    1. Writing Files
    • Synchronous File Writing: Writes data to a file in a blocking manner, pausing execution until the write operation completes.

    Example:# 

    import 'dart:io';

void main() async {
await File('example.txt').writeAsString('Hello, Dart!');
}
    1. Deleting Files
    • Deleting a File: Removes a specified file from the file system.

    Example:# 

    import 'dart:io';

void main() {
  File('example.txt').deleteSync();
}
    1. Creating and Renaming Files Creating Files: Create a new file or open an existing one if it already exists.

    Example:# 

    import 'dart:io';

void main() {
  File('newfile.txt').createSync();
}
    • Renaming Files: Rename or move a file from one name to another.

    Example:

    import 'dart:io';

void main() {
  File('oldname.txt').renameSync('newname.txt');
}
    1. Checking File Existence File Existence: Determine whether a file exists in the file system.

    Example:# 

    import 'dart:io';

void main() {
  bool exists = File('example.txt').existsSync();
  print('File exists: $exists');
}
    1. Working with Directories Creating Directories: Create new directories in the file system.

    Example:# 

    import 'dart:io';

void main() {
  Directory('newdir').createSync();
}
    • Listing Directory Contents: List all files and subdirectories within a specified directory.

    Example:# 

    
import 'dart:io';
void main() {
  var dir = Directory('mydir');
  dir.listSync().forEach((entity) {
    print(entity.path);
  });
}
- Deleting Directories: Remove a directory and optionally its contents.
 
 ### Example:
```dart
import 'dart:io';

void main() {
  Directory('oldir').deleteSync(recursive: true);
}
    1. Handling Exceptions Error Handling: Use try-catch blocks to manage and handle errors that may occur during file operations.

    Example:# 

    import 'dart:io';

void main() {
  try {
    File('example.txt').readAsStringSync();
  } catch (e) {
    print('An error occurred: $e');
  }
}
    1. Working with File Paths
    • Relative and Absolute Paths: Understand and manipulate file paths for file operations.

    Example:# 

    import 'dart:io';

void main() {
  String relativePath = 'folder/file.txt';
  String absolutePath = Directory.current.absolute.path + '/' + relativePath;
  print('Absolute Path: $absolutePath');
}
    • Path Manipulation: Use the path package to handle more complex path operations.

    Example:# 

    import 'package:path/path.dart' as p;

void main() {
  String fullPath = p.join('folder', 'file.txt');
  print('Full Path: $fullPath');
}
    1. File Encoding Handling Encodings: Read and write files with different encodings, such as UTF-8.

    Example:# 

    import 'dart:io';
import 'dart:convert';

void main() {
  File('example.txt').writeAsStringSync('Hello, Dart!', encoding: utf8);
  String content = File('example.txt').readAsStringSync(encoding: utf8);
  print(content);
}
    1. File Handling Best Practices

    • Ensuring File Closure: Always ensure that file resources are properly closed after operations to prevent resource leaks.

    • Managing Large Files: Handle large files efficiently by processing data in chunks to avoid high memory usage.

    Example:

    import 'dart:io';

void main() async {
  final file = File('largefile.txt');
  final sink = file.openWrite();

  // Write data in chunks
  for (var i = 0; i < 1000; i++) {
    sink.write('Line $i\n');
  }

  await sink.flush();
  await sink.close();
}
