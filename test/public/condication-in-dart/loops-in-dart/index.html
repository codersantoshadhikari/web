    Loops in Dart#

    Types of Loops#

    Dart provides several types of loops to repeatedly execute a block of code:

    1. for Loop: Iterates a set number of times.
    2. for-in Loop: Iterates over each element in a collection, such as a list or set.
    3. while Loop: Repeats a block of code as long as a specified condition is true.
    4. do-while Loop: Similar to the while loop, but the condition is checked after the block of code has been executed.

    How They Work#

    • for Loop: Executes a block of code a fixed number of times, based on a counter variable.
    • for-in Loop: Iterates through each item in a collection, such as a list or set, making it easy to traverse elements.
    • while Loop: Continues to execute a block of code as long as the given condition is true.
    • do-while Loop: Executes the block of code once before checking the condition, ensuring that the loop runs at least once.

    Overall Usage#

    Loops are used to automate repetitive tasks in your code, allowing you to perform operations multiple times without having to write the same code repeatedly. Each type of loop has its own use case, depending on whether you need to iterate a fixed number of times, traverse a collection, or continue until a condition is no longer true.

    Examples#

    for Loop# 

    void main() {
  for (int i = 0; i < 5; i++) {
    print('Iteration $i');
  }
}

    for-in Loop# 

    void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];

  for (String fruit in fruits) {
    print(fruit);
  }
}

    while Loop# 

    void main() {
  int count = 0;

  while (count < 5) {
    print('Count is $count');
    count++;
  }
}

    do-while Loop# 

    void main() {
  int count = 0;

  do {
    print('Count is $count');
    count++;
  } while (count < 5);
}

    Overall#

    Loops in Dart allow you to execute a block of code multiple times, automating repetitive tasks. The for and for-in loops are great for iterating a specific number of times or over collections, while the while and do-while loops continue executing as long as a condition is true. Each loop type has its own use case, making them powerful tools for controlling the flow of your program.
